# 📋 Семинар - Система регистрации участников

Веб-приложение для регистрации участников семинара с админ-панелью для управления данными.

## ✨ Функции

### Для участников
- 📝 **Регистрация** - ввод ФИО и учебного заведения
- ✅ **Подтверждение** - мгновенная подтверждение регистрации
- 📊 **Счётчик** - просмотр количества зарегистрированных участников

### Для администраторов
- 🔐 **Защита паролем** - безопасный доступ к панели управления
- 👥 **Список участников** - просмотр всех зарегистрированных
- 🔍 **Поиск** - быстрый поиск по ФИО
- 📈 **Сортировка** - по ФИО и названию учебного заведения
- ✔️ **Отслеживание** - отметить присутствующих на семинаре
- 📥 **Экспорт** - скачивание списка в формате CSV для Excel

## 🚀 Быстрый старт

### Локальное использование
1. Скачайте `index.html`
2. Откройте файл в браузере
3. Готово! 🎉

### GitHub Pages (онлайн)
1. Перейдите в **Settings** репозитория
2. Выберите **Pages**
3. Выберите ветку `main`
4. Сохраните
5. Сайт будет доступен по адресу: `https://aknazarovasuita-code.github.io/seminar-registration/`

## 🔐 Админ-панель

**Доступ:** Нажмите кнопку в браузере (или введите `#admin` в URL)

## 💾 Хранение данных

- Данные хранятся в **localStorage** браузера
- Сохраняются автоматически
- Не требует сервера
